# 05 数据流（去重核心）

**参考04模块**：录入→查询→分配→统计流程。

## 录入去重流 (recorder)
```
输入号码list → 查DB exists → 分支:
  新增: insert + batch_id → {'new':n}
  重复: collect list → {'dup':n, 'dups':list}
→ 日志 + 返回dict
```

## 查询流
```
phone → SELECT * → merge assign/stats → df/json
```

## 注意事项
- 事务: conn.commit() per batch.
- 边界: empty list → {}
- 扩展: 异步批量 (threading)。

**检查**：流程无环，DB锁预防 (timeout)。

**防护**：号码规范11位数字，统一trim()。
