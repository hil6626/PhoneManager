# 01 项目总览

## 项目目标
构建 Windows 本地小程序，实现号码数据去重录入、查询、分配、统计报表。核心价值：自动去重分析，提高号码利用效率，防止重复浪费。

## 核心功能（4大模块）
- **智能录入+去重**：单/批量录入，输出新增/重复统计，只保存新增。
- **号码查询**：单/批量，显示生命周期（来源、重复次数、分配）。
- **分类分配**：推广/进线/金额/日期，支持批量。
- **报表统计**：时间/批次统计，去重率，Excel/CSV导出。

## 技术栈
- **GUI**：Python 3.11+ + PyQt5
- **数据**：SQLite (data.db)，pandas 处理
- **报表**：openpyxl (Excel)
- **打包**：pyinstaller → .exe
- **开发**：Ubuntu 24 + VSCode → Windows 部署

## 项目价值与范围
- **单机本地**，数据量万级，支持扩展多人(MySQL)。
- **生产可扩展**：日志监控、备份、权限。
- **难度**：初学者 1周开发（分模块）。
