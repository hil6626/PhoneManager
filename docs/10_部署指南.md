# 10 部署指南

## Ubuntu dev (参考 docs/06_环境规划)

### 1. 环境准备
```bash
cd PhoneManager
python3 -m venv venv  # 或 tmpvenv 测试
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt  # PyQt5 pandas openpyxl pyinstaller
deactivate
```

### 2. 批量导入测试 (add_batch)
```bash
cd PhoneManager
source venv/bin/activate  # 或 tmpvenv
python3 -c "
from modules.recorder import Recorder
r = Recorder()
print(r.add_batch('test_data/test_numbers.xlsx'))
# 预期: {'new_count':4, 'dup_count':1, 'dup_numbers':['138...'], 'file':'...'}
"

# CSV 测试类似
print(r.add_batch('test_data/test_numbers.csv'))
```

### 3. DB 验证
```bash
sqlite3 database/data.db "
SELECT COUNT(*) FROM numbers;
SELECT number, source FROM numbers WHERE source LIKE '%test%' ORDER BY created_at DESC LIMIT 5;
"
# 检查 new_count == 新增 COUNT, dup_numbers 已存在不重复
```

### 6. EXE打包&测试 (Windows, portable)
**前提**: Python3.11+venv, requirements.txt deps.

```bash
cd PhoneManager
# 生成spec (onefile+windowed)
pyi-makespec --onefile --windowed --name PhoneManager --icon=resources/icons/app.ico --add-data "config;config" --add-data "database;database" --add-data "resources;resources" --hidden-import=pandas --hidden-import=numpy --hidden-import=openpyxl main.py
# 编辑main.spec: 分析=Analysis(..., datas=[('config','config'),...])
pyinstaller main.spec  # ~2min构建
```

**验证** (dist/PhoneManager.exe 双击GUI):
1. **导入**: 选test_data/test_numbers.xlsx → 预期{'new_count':X, 'dup_count':Y}
2. **查询**: lookup_batch → processed/dups统计
3. **分配**: assign_batch 'sales' → success/fail
4. **全链路**: sqlite3 database/data.db "SELECT COUNT(*)"; tail -f logs/app.log

**分发**: dist/PhoneManager.exe + test_data/ (离线OK)

**注意**:
- Binary fix: hidden-imports防numpy DLL miss
- 大文件: --exclude-module matplotlib (未用)
- 检查: pyinstaller --debug=all 排查
