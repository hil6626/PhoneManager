Windows小程序功能说明文档（号码管理系统）
一、总体描述

本程序旨在帮助用户在Windows环境下高效管理号码数据，提供录入、查询、分配和统计功能。程序需具备操作简便、数据准确、报表清晰的特点。

二、功能模块
1. 号码录入模块

功能描述：

支持单条录入和批量导入（Excel/CSV）。
每条号码自动记录：
录入时间（精确到秒）
来源文档
录入人（可选）
自动检查号码是否重复：
新号码：保存成功
重复号码：提示“已存在”，可选择覆盖或跳过
提供录入日志，显示最近录入记录

界面设计：

输入框：号码、来源文档、录入人
按钮：录入/导入/清空
表格显示：
号码 | 录入时间 | 来源文档 | 录入人 | 重复状态

功能流程：

用户输入号码 → 点击“录入”
系统检查数据库：
若号码存在 → 弹出提示
若新号码 → 保存并显示成功
支持批量导入：
选择文件 → 系统逐条检查 → 生成导入报告
2. 号码查询模块

功能描述：

单号查询或批量查询
显示号码归属信息：
所属文档
录入时间
分配类型、进线日期、金额（如有）
支持模糊查询、部分匹配

界面设计：

输入框：号码/关键词
按钮：查询
表格显示：
号码 | 文档名称 | 录入时间 | 分配类型 | 进线日期 | 金额

功能流程：

用户输入号码 → 点击“查询”
系统搜索数据库 → 返回查询结果
用户可导出查询结果（Excel/CSV）
3. 号码分配模块

功能描述：

对号码进行分类分配
支持多种类型：
推广名单
进线号码
进线日期
金额
可对已录入号码批量分配
可修改分配信息

界面设计：

多选框选择号码
下拉框选择分配类型
日期选择器：进线日期
金额输入框
按钮：保存分配信息

功能流程：

用户选择号码 → 设置分配信息 → 点击保存
系统更新数据库
已分配号码可在查询模块查看
4. 报表统计模块

功能描述：

按时间段统计号码信息：
新录入号码数
重复号码数
各类型分配数量
金额统计
支持导出Excel/CSV报表
自动生成统计表（可设置定时或手动）

界面设计：

日期选择器：开始时间/结束时间
按钮：生成报表 / 导出报表
表格显示统计结果：
时间段 | 新录入数 | 重复数 | 各类型数量 | 金额总计

功能流程：

用户选择时间段 → 点击“生成报表”
系统统计并显示结果
用户可导出报表，便于二次处理
三、数据库设计
1. 表：号码信息表
字段名	类型	描述
ID	自增主键	唯一标识
号码	字符串	唯一号码
录入时间	datetime	录入时间
文档名称	字符串	来源文档名称
录入人	字符串	可选录入人员
分配类型	字符串	推广/进线等分类
进线日期	date	可选进线日期
金额	decimal	可选金额
2. 表：分配类型表
字段名	类型	描述
类型ID	主键	唯一标识
类型名称	字符串	分类名称
3. 表：报表历史表（可选）
字段名	类型	描述
报表ID	主键	唯一标识
生成时间	datetime	报表生成时间
时间范围	字符串	统计时间段
统计结果路径	字符串	报表文件路径
