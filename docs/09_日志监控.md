# 09 日志监控

**logs/app.log** rotating.

## 级别
- INFO: 录入success {'batch':id, 'new':72}
- WARNING: dup high (>50%)
- ERROR: DB lock/input invalid

## 示例code (utils/log.py)
```python
import logging
logging.basicConfig(filename='logs/app.log', level=logging.INFO, 
rotatingFileHandler(maxBytes=10*1024*1024))
```

**监控**：tail -f logs/app.log | grep ERROR

**Phase5监控** (exe日志):
```bash
tail -f logs/app.log | grep -E "导入完成|dup high|ERROR|分配成功"
# 预期: %(asctime)s [INFO] 导入完成 {'new_count':72}
```
rotatingFileHandler(maxBytes=10MB, backup=5).
**防护**：no PII log (hash phone?), daily rotate.
