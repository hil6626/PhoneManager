# 06 环境规划（开发/生产）

**参考02目录**：Ubuntu开发 → Windows部署，确保兼容。

## 1. 开发环境 (Ubuntu24+)
```
cd PhoneManager
python3 -m venv venv  # 隔离
source venv/bin/activate
pip install -r requirements.txt
pip check  # 依赖OK
```

## 2. 测试环境验证
```bash
python -c "import PyQt5, pandas, openpyxl; print('OK')"
```

## 3. 生产打包 (Windows)
- Install Python3.11
- pip install -r requirements.txt pyinstaller
- `pyinstaller --onefile --windowed --icon=resources/icons/app.ico main.py`
- dist/main.exe (portable)

## 4. DB/日志
- database/data.db auto-create
- logs/app.log (rotating, <10MB)

## 注意事项
- Python固定3.11, no global pip.
- 扩展: Docker compose for server版.
- 冲突避免: venv deactivate清理.

**检查点**：venv/bin/activate存在, import无错.
**防护**：pip freeze > locked.txt锁定版本.
**难度**：低, 5min setup.
