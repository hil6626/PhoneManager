# 12 运维手册

**日常运维**：
- `tail -f PhoneManager/logs/app.log | grep ERROR`
- `cp database/data.db backup_$(date +%Y%m%d).db`
- DB维护：`sqlite3 database/data.db "VACUUM; PRAGMA integrity_check;"`

**升级流程**：
1. git pull origin main (PhoneManager/)
2. 06环境重装依赖
3. pyinstaller repack
4. 测试batch → deploy exe

**监控指标**：
- 去重率>30% → 查来源文档质量 (13问题)
- log >1GB → rotate

**扩展**：cron backup脚本.

**Phase5运维** (exe/DB):
```bash
# 备份/优化
cp database/data.db backup_$(date +%F).db
sqlite3 database/data.db "VACUUM; CREATE INDEX IF NOT EXISTS idx_number ON numbers(number); PRAGMA integrity_check;"
# 升级: git pull → pyinstaller → test batch
```
**监控**: 去重率= dup_count/total >30% → 查test_data质量 (docs/13).
难度低, 自动化优先.
