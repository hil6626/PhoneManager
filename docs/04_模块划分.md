# 04 模块划分（职责&依赖）

**参考03 API**：细化modules/文件职责，图示依赖。

## 模块职责表
| 模块 | 职责 | 输入/输出 | 依赖 |
|------|------|-----------|------|
| recorder.py | 去重录入/批次 | list[str] → dict{new/dup} | sqlite3, pandas |
| query.py | 查询/导出df | str/list → dict/df | recorder (读DB) |
| assigner.py | 分配更新 | dict → bool/int | query (查前) |
| stats.py | 统计/报表 | date_range → df | all (聚合) |

## 依赖图 (文本)
```
main_window.py --> ui/widgets + modules/*
modules/* --> database/data.db (事务)
```

## 注意
- 无循环依赖。
- 每个py独立测试：python recorder.py test_data.csv
- 扩展：加 utils/db.py 公用conn。

**操作流程示例** (同步main.py/main.py GUI):
```python
# CLI测试 (Phase5)
from modules.recorder import Recorder; r=Recorder(); print(r.add_batch('test_data/test_numbers.csv'))  # {'new':4,'dup':1}
```
