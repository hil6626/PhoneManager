# 07 安全规范

**本地单机**，焦点数据完整/权限。

## 规则
- DB: UNIQUE phone, backup daily.
- 输入: sanitize phone (re.match(r'1[3-9]\d{9}$'))
- 日志: no敏感info (hash phone?)
- 权限: 文件mode 600 data.db
- 异常: try-except all DB/io, log+user msg.

## 检查
```bash
chmod 600 PhoneManager/database/*
sqlite3 data.db "PRAGMA integrity_check;"
```

**防护**：no sql injection (parametrized), no exec input.
**扩展**：user/pass for multi.

难度低.
